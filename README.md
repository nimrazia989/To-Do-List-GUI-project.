# 📝 To-Do List GUI App (Python)

A simple and modern To-Do List application built using Python and Tkinter.
This project is designed for beginners to practice GUI development and basic Python concepts.

---

## 🚀 Features

* ✅ Add new tasks
* ❌ Delete selected tasks
* 🧹 Clear all tasks
* 🎨 Simple and clean graphical interface

---

## 🛠️ Technologies Used

* Python
* Tkinter (built-in GUI library)

---

## ▶️ How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Run the following command:

python main.py
